What is PostgreSQL?

Postgresql is an open source relational database that is often used in development. It runs on all major operating systems, including Unix, Linux, and Windows. It is largely compliant to the ANSI:SQL 2008 standard, which is unique of most databases. Although it is not quite as popular as some alternatives, such as MySQL, it is more customizable and often more reliable.

The Background

Postgresql was developed by a group of volunteers in the Postgresql Global Development Group, but it also has many extensions that were contributed by third parties. The name originally came from a post-Ingres database, where post-Ingres comes from INteractive Grahics REtrieval System. While PostgreSQL originally started in 1982 at the University of California Berkeley, it has changed and evolved over the years, adding SQL support and improving. Now there are updates about once a year from people around the world.

Features

PostgreSQL has great features that can make web development easier.
  • a wide variety of data types
  • user-defined objects
  • triggers, which cause an action on the database based on an INSERT or UPDATE statement
  • manages concurrency issues based on MVCC, which is multiversion concurrency control. More info about MVCC can be found here.
  • easily compatible with other web frameworks, such as Django, Drupal, .NET
