Welcome.

The World Wide Web revolutionized communication and the way we interact with information on a global level. Two of the most fundamental Web technologies that continue to influence the way we consume and process information online are Hypertext Markup Language, or HTML, and Cascading Style Sheets, or CSS. This Website focuses on the role of HTML5 and CSS3 in modern Web development, with particular emphasis on mobile browsing and Responsive Web Design.

A brief history of the World Wide Web

Before getting into the history of the World Wide Web, we need to start by explaining the Internet.

The Internet

The Internet is a super-duper huge system of networks; a network of networks. A network is a collection of two or more computers that are linked together in such a way that enables communication between them. The Internet uses standardized methods of communication to exchange information between computers (hosts) and networks.

The World Wide Web

The WWW is a collection of documents that are accessed through the Internet. These documents are written in hypertext - a special type of text designed for computers, which is also used to describe/define links to other documents, images, and other media. Hypertext is powerful because of its structure and hidden elements, which control the display of information, and connection to other documents.

In short, the Internet and WWW is an enormous electronic system of files that allow people like you and me to access and view it in pretty formats that we can read. Hypertext Markup Language and Cascading Style Sheets help us to access and read that information in a way we can understand.

Mobile computing and Responsive Web Design

Advancements in computing technologies put the World Wide Web in people's pockets. Today, the wide variety of screen sizes and resolutions, along with varying browsers and hardware capabilities, make Web design and development an even more interesting task. In order to respond to these changes (no pun intended), the concept of Responsive Web Design came about.

Responsive Web Design addresses the issue of varying screen sizes and a wide range of potential visitors by using Web technologies to serve up or dynamically present content on Web pages. To learn more about Responsive Web Design (or RWD), click on the link below.
